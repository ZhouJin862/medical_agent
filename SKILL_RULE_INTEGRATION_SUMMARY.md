# 技能与规则引擎集成 - 完整使用示例

## 系统概述

成功实现了技能(Skills)与规则引擎(Rules)的深度集成，创建了"AI + 规则"混合决策架构：

```
用户输入 → 技能识别 → 规则评估 → 增强提示词 → LLM响应 → 用户收到准确建议
```

## 已创建的生产环境技能

### 1. 高血压评估 (hypertension_assessment)
- **描述**: 评估患者血压水平，根据临床指南进行分级诊断
- **规则配置**:
  - 规则分类: diagnosis, risk_assessment
  - 关联病种: hypertension
  - 生命体征评估: 启用
  - 风险评分: 启用
- **意图关键词**: 高血压, 血压, 血压高, 收缩压, 舒张压

### 2. 糖尿病评估 (diabetes_assessment)
- **描述**: 评估患者血糖水平，进行糖尿病筛查和风险评估
- **规则配置**: diagnosis, risk_assessment, reference_value
- **意图关键词**: 糖尿病, 血糖, 高血糖, 空腹血糖

### 3. 血脂异常评估 (dyslipidemia_assessment)
- **描述**: 评估患者血脂水平，识别高脂血症风险
- **规则配置**: diagnosis, risk_assessment, reference_value
- **意图关键词**: 血脂, 胆固醇, 甘油三酯, 高血脂

### 4. 痛风评估 (gout_assessment)
- **描述**: 评估患者尿酸水平，识别痛风风险
- **规则配置**: diagnosis, risk_assessment, reference_value
- **意图关键词**: 痛风, 尿酸, 关节痛, 高尿酸

### 5. 健康体检综合评估 (health_checkup_assessment)
- **描述**: 综合分析体检数据，评估"四高"风险
- **规则配置**: diagnosis, risk_assessment, reference_value
- **意图关键词**: 体检, 健康体检, 体检报告

### 6. 肥胖评估 (obesity_assessment)
- **描述**: 评估患者BMI和体脂情况，识别肥胖风险
- **规则配置**: diagnosis, risk_assessment
- **意图关键词**: 肥胖, 体重, BMI, 减肥, 超重

### 7. 用药提醒 (medication_reminder)
- **描述**: 提醒患者按时服药，解释药物作用
- **规则配置**: prescription
- **意图关键词**: 吃药, 用药, 服药, 药物

### 8. 健康咨询 (health_consultation)
- **描述**: 回答健康问题，提供基础健康指导
- **规则配置**: risk_assessment
- **意图关键词**: 健康, 咨询, 建议, 指导

### API调用示例

```bash
# 评估高血压 (BP: 150/95)
curl -X POST "http://localhost:8006/api/skills/{skill_id}/evaluate-with-rules" \
  -H "Content-Type: application/json" \
  -d '{
    "patient_id": "patient_001",
    "user_input": "我血压150/95，属于什么级别？",
    "extracted_data": {
      "systolic": 150,
      "diastolic": 95,
      "age": 50,
      "gender": "male"
    }
  }'
```

```
│         (匹配关键词: "血压" → hypertension_assessment)         │

                             ▼
┌─────────────────────────────────────────────────────────────┐
│                    规则评估层                                    │

                             │

┌─────────────────────────────────────────────────────────────┐
│                   增强提示词生成                                   │

│  基于上述临床规则，提供准确的医疗建议...                          │
└────────────────────────────┬────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────┐
│                    LLM响应生成                                   │
│  "根据临床规则，您的血压150/95属于高血压1级..."                  │
└─────────────────────────────────────────────────────────────┘
```

## 服务器状态

- **后端服务器**: http://localhost:8006
- **技能数量**: 8个生产环境技能
- **规则数量**: 4条高血压相关规则

## 技术栈

- **后端**: FastAPI + Python 3.13
- **前端**: React + TypeScript + Material-UI
- **数据库**: MySQL (aiomysql)
- **规则引擎**: 自定义规则评估引擎
- **AI模型**: 集成GLM-5
