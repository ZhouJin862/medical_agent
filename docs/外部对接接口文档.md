## 1. 同步洞察结果接口

### 基本信息
- **接口名称**: 新增洞察数据
- **消费方id**: `1abc`
- **api编码**: insightSave
- **功能描述**: 同步健康评估的洞察结果数据

### 请求参数

#### 请求体 (JSON)
```json
{"party_id":"12311112","risk_level":"medium","population_classification":{"primary_category":"亚健康","grouping_basis":[{"items":null}]},"abnormal_indicators":{"indicators":[{"items":null}],"warnings":[{"items":null}]},"recommended_data_collection":[{"items":null}],"disease_prediction":[{"items":null}],"intervention_prescriptions":[{"items":null}],"risk_warnings":[{"items":null}],"assessed_at":"2026-04-29 14:55:00","recommended_target":[{"items":"推荐目标"}]}
```

#### 参数说明
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| party_id | String | 是 | 用户ID |
| risk_level | String | 是 | 风险等级 (low/medium/high) |
| population_classification | Object | 是 | 人群分类信息 |
| population_classification.primary_category | String | 是 | 主要分类 (如: 亚健康) |
| population_classification.grouping_basis | Array | 是 | 分组依据 |
| abnormal_indicators | Object | 是 | 异常指标信息 |
| abnormal_indicators.indicators | Array | 是 | 指标列表 |
| abnormal_indicators.warnings | Array | 是 | 警告列表 |
| recommended_data_collection | Array | 是 | 推荐数据采集 |
| disease_prediction | Array | 是 | 疾病预测 |
| intervention_prescriptions | Array | 是 | 干预处方 |
| risk_warnings | Array | 是 | 风险警告 |
| assessed_at | String | 是 | 评估时间 |
| recommended_target | Array | 是 | 推荐目标 |

### 响应参数

#### 响应参数说明
| 参数名 | 类型 | 说明 |
|--------|------|------|
| reqId | String | 请求ID |
| time | String | 请求时间 |
| sysTime | String | 系统时间 |
| ret | String | 返回码 (0: 成功) |
| code | String | 状态码 (0: 成功) |
| msg | String | 返回消息 |
| message | String | 详细消息 |
| data | Object | 返回数据 |

﻿## 2. 更新健康信息接口

### 基本信息
- **接口名称**: 添加健康信息
- **消费方id**: `1abc`
- **api编码**: addHumanInfo
- **功能描述**: 更新或添加用户的健康数字人信息

### 请求参数

#### 请求体 (JSON)
```json
{"healthInsightAddDTO":{"healthDigitalHumanDTO":{"age":30,"gender":"男","height":"175","weight":"70","waistCircumference":"80","healthDigitalBloodPressureDTO":{"systolicPressure":"120","diastolicPressure":"80"},"healthDigitalUricAcidDTO":{"bloodUricAcid":"350"},"healthDigitalBloodLipidsDTO":{"hbalc":"5.0","tc":"4.5","tg":"1.5","ldlc":"2.5","hdlc":"1.2"},"healthDigitalBloodGlucoseDTO":{"breakfastPostprandialGlucose":"6.0","oneHourPostprandialGlucose":"8.0","preLunchGlucose":"5.5","postLunchGlucose":"7.5","preDinnerGlucose":"5.8","postDinnerGlucose":"7.2","randomGlucose":"6.5","bedtimeGlucose":"5.5","earlyMorningGlucose":"5.0","twoHourPostprandialGlucose":"6.8","fastingGlucose":"5.2"},"healthDigitalDiseaseDTO":{"chronicDiseaseDTOList":[{"icdCode":"I10","icdName":"原发性高血压"}],"specialDiseaseDTOList":[{"icdCode":"E11","icdName":"2型糖尿病"}]},"questionnaireInfoList":[{"question":"您是否有高血压病史？","answer":"是"}]}}}
```
﻿#### 参数说明

##### 基础信息
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| age | Integer | 是 | 年龄 |
| gender | String | 是 | 性别 (男/女) |
| height | String | 是 | 身高 (cm) |
| weight | String | 是 | 体重 (kg) |
| waistCircumference | String | 是 | 腰围 (cm) |

##### 血压信息 (healthDigitalBloodPressureDTO)
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| systolicPressure | String | 是 | 收缩压 (mmHg) |
| diastolicPressure | String | 是 | 舒张压 (mmHg) |

##### 尿酸信息 (healthDigitalUricAcidDTO)
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| bloodUricAcid | String | 是 | 血尿酸 (μmol/L) |

﻿##### 血脂信息 (healthDigitalBloodLipidsDTO)
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| hbalc | String | 是 | 糖化血红蛋白 (%) |
| tc | String | 是 | 总胆固醇 (mmol/L) |
| tg | String | 是 | 甘油三酯 (mmol/L) |
| ldlc | String | 是 | 低密度脂蛋白胆固醇 (mmol/L) |
| hdlc | String | 是 | 高密度脂蛋白胆固醇 (mmol/L) |

##### 血糖信息 (healthDigitalBloodGlucoseDTO)
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| breakfastPostprandialGlucose | String | 是 | 早餐后血糖 (mmol/L) |
| oneHourPostprandialGlucose | String | 是 | 餐后1小时血糖 (mmol/L) |
| preLunchGlucose | String | 是 | 午餐前血糖 (mmol/L) |
| postLunchGlucose | String | 是 | 午餐后血糖 (mmol/L) |
| preDinnerGlucose | String | 是 | 晚餐前血糖 (mmol/L) |
| postDinnerGlucose | String | 是 | 晚餐后血糖 (mmol/L) |
| randomGlucose | String | 是 | 随机血糖 (mmol/L) |
| bedtimeGlucose | String | 是 | 睡前血糖 (mmol/L) |
| earlyMorningGlucose | String | 是 | 凌晨血糖 (mmol/L) |
| twoHourPostprandialGlucose | String | 是 | 餐后2小时血糖 (mmol/L) |
| fastingGlucose | String | 是 | 空腹血糖 (mmol/L) |

##### 疾病信息 (healthDigitalDiseaseDTO)
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| chronicDiseaseDTOList | Array | 否 | 慢性病列表 |
| chronicDiseaseDTOList[].icdCode | String | 是 | ICD编码 |
| chronicDiseaseDTOList[].icdName | String | 是 | 疾病名称 |
| specialDiseaseDTOList | Array | 否 | 特殊疾病列表 |
| specialDiseaseDTOList[].icdCode | String | 是 | ICD编码 |
| specialDiseaseDTOList[].icdName | String | 是 | 疾病名称 |

##### 问卷信息 (questionnaireInfoList)
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| questionnaireInfoList | Array | 否 | 问卷信息列表 |
| questionnaireInfoList[].question | String | 是 | 问题 |
| questionnaireInfoList[].answer | String | 是 | 答案 |
