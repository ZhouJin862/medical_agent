## 快速入门

### 1. 系统简介

Medical Agent 是一个基于人工智能的慢病健康管理助手，能够帮助您：

- 📊 **健康评估**：实时分析血压、血糖、血脂等健康指标
- 🎯 **风险预测**：提前预警慢性病风险
- 📋 **个性化处方**：生成饮食、运动、睡眠等健康处方
- 💬 **智能对话**：7x24小时健康咨询
- 📈 **健康追踪**：记录和管理您的健康数据

### 2. 首次使用

#### 步骤 1：注册账号

```bash
# 访问系统
https://medical-agent.example.com

# 点击"注册"
# 填写手机号和验证码
# 设置密码
```

#### 步骤 2：完善个人信息

登录后，请完善以下信息：

```

## 核心功能

### 1. 智能对话

#### 如何提问

您可以用自然语言提问，例如：

```
"我的血压是135/88，这正常吗？"
"我空腹血糖6.5，有糖尿病风险吗？"
"体检发现尿酸高，我应该注意什么？"
```

#### 对话技巧

**✅ 推荐做法：**

- 提供具体数值（如血压 135/88）
- 说明测量条件（如空腹血糖）
- 表达清晰的问题

**❌ 避免做法：**

- 过于笼统的描述（"我身体不好"）
- 多个问题混杂在一起
- 提供非医学测量数据

### 2. 健康评估

#### 支持的评估项目

| 项目 | 输入示例 | 评估内容 |
|------|----------|----------|
| 血压 | 135/88 | 高血压等级、心血管风险 |
| 血糖 | 空腹 6.5 | 糖尿病风险、代谢状态 |
| 血脂 | TC 6.5 | 血脂异常、动脉硬化风险 |
| 尿酸 | 450 μmol/L | 痛风风险、关节健康 |
| 体重 | BMI 28 | 肥胖等级、代谢风险 |

#### 查看评估结果

评估结果包含：

```
📊 指标等级：正常高值
⚠️ 风险分类：低风险
📋 详细解读：
  - 收缩压略高于正常值
  - 舒张压在正常范围内
  - 建议定期监测
💡 健康建议：
  - 低盐饮食
  - 适量运动
  - 控制体重
```

### 3. 风险预测

#### 风险类型

系统可以预测以下风险：

| 风险类型 | 说明 |
|----------|------|
| 心血管疾病 | 10年心血管疾病风险 |
| 糖尿病 | 糖尿病发病风险 |
| 代谢综合征 | 代谢综合征风险 |
| 脑卒中 | 脑卒中发病风险 |

#### 风险等级

- 🟢 **低风险**：风险 < 10%
- 🟡 **中风险**：风险 10-20%
- 🟠 **高风险**：风险 20-30%
- 🔴 **极高风险**：风险 > 30%

### 4. 健康处方

#### 处方类型

##### 4.1 饮食处方

```
🥗 DASH 饮食方案

推荐食物：
✅ 蔬菜类：500-600克/天
✅ 水果：200-300克/天
✅ 低脂乳制品：2-3份/天
✅ 全谷物：6-8份/天

限制食物：
❌ 钠盐：< 5克/天
❌ 红肉：< 3份/周
❌ 添加糖：< 25克/天
❌ 饱和脂肪：< 10% 总热量
```

##### 4.2 运动处方

```
🏃 有氧运动处方

运动类型：
- 快走
- 游泳
- 骑自行车
- 跳舞

运动频率：每周 5 次
运动时长：30-45 分钟/次
运动强度：中等强度

注意事项：
- 从低强度开始
- 循序渐进
- 运动前后热身
```

##### 4.3 睡眠处方

```
😴 改善睡眠处方

睡眠目标：每晚 7-8 小时

睡眠卫生：
- 固定作息时间
- 睡前 1 小时不看电子屏幕
- 保持卧室安静、黑暗、凉爽
- 避免睡前饮酒和咖啡因

睡眠改善技巧：
- 4-7-8 呼吸法
- 渐进式肌肉放松
- 正念冥想
```

### 5. 健康数据管理

#### 记录体征数据

可以通过以下方式记录：

1. **对话中记录**
   ```
   "我今天的血压是128/82"
   "空腹血糖测得5.6"
   ```

2. **批量录入**
   ```
   批量录入过去一周的血压数据：
   - 周一：130/85
   - 周三：125/82
   - 周五：128/83
   ```

3. **数据导出**
   - 导出为 PDF 报告
   - 导出为 Excel 表格
   - 分享给医生

系统会自动生成趋势图表：

## 常见使用场景

### 场景 1：高血压管理

#### 用户故事

> 我是高血压患者，每天都要测量血压。以前都是自己记录在本子上，看不到趋势。现在我可以直接告诉系统我的血压值，系统会帮我分析、画趋势图，还给我建议。

#### 使用流程

```
1. 每天测量血压后，打开对话
2. 说："我的血压是130/85"
3. 系统自动记录并分析
4. 查看反馈建议
5. 定期查看趋势报告
```

#### 系统提供的帮助

- 📊 自动绘制血压趋势图
- 🎯 评估控制目标达成情况
- 💡 提供个性化的生活方式建议
- ⚠️ 异常值自动预警

### 场景 2：糖尿病预防

#### 用户故事

> 我有糖尿病家族史，担心自己也会得糖尿病。系统帮我评估了当前的风险，并给出了具体的预防建议，包括饮食、运动、作息等各个方面。

#### 使用流程

```
1. 输入空腹血糖和糖化血红蛋白数据
2. 系统评估糖尿病风险
3. 获取详细的预防指南
4. 制定个人健康计划
5. 定期复查和跟踪
```

#### 预防建议包括

- 🍎 饮食调整建议
- 🏃 运动方案设计
- 😴 睡眠质量改善
- 📅 定期检查提醒

### 场景 3：综合健康管理

#### 用户故事

> 我有多种慢性病风险因素，包括高血压前期、糖尿病前期、超重等。系统能够综合考虑我的所有健康数据，给我一个全面的健康画像和综合管理方案。

#### 系统能力

- 🎯 综合评估四高一重风险
- 📋 生成综合健康计划
- 📊 可视化健康雷达图
- 🔄 动态调整管理方案

### 场景 4：用药咨询

#### 用户故事

> 医生给我开了新的降压药，我想知道它和现在的药物会不会有冲突。系统可以帮我检查药物相互作用，并给出安全建议。

#### 功能说明

⚠️ **重要提醒：** 系统提供的药物信息仅供参考，不能替代专业医疗建议。如有疑问，请务必咨询医生或药师。

```

建议立即行动：
1. 保持冷静，坐下休息

## 常见问题

### Q1：我的数据安全吗？

**A:** 是的，我们非常重视数据安全：

- 🔒 所有数据传输使用 HTTPS 加密
- 🔐 敏感信息加密存储
- 🛡️ 通过安全认证和授权
- 📜 符合医疗数据保护法规

### Q2：系统的建议可以替代医生吗？

**A:** 不可以。系统的作用是：

- ✅ 健康知识普及
- ✅ 健康风险评估
- ✅ 生活方式指导
- ✅ 求医参考建议

❌ **不能替代：**
- 医学诊断
- 药物处方
- 手术治疗
- 急救处置

### Q3：如何确保结果准确？

**A:** 系统准确性保障：

- 基于权威医疗指南（如《中国高血压防治指南》）
- 使用经过验证的评估标准
- 定期更新医学知识库
- 专业医疗团队审核

**建议：**
- 定期与实际检查数据对比
- 有疑问时咨询医生
