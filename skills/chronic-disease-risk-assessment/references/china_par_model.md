## 模型概述

### 什么是China-PAR模型？

China-PAR（Prediction for ASCVD Risk in China）是中国人群心血管病风险评估模型，由中美科学家联合开发，基于中国人群大规模前瞻性队列研究数据构建。

### 模型特点
- **专为中国人群设计**：基于中国本土数据
- **预测准确**：在中国人群中具有良好的预测能力
- **临床实用**：参数易获取，计算简便

### 预测目标
- **10年ASCVD发病风险**：动脉粥样硬化性心血管疾病
- **ASCVD包括**：急性心肌梗死、冠心病死亡、脑卒中

## 预测因子

China-PAR模型包含以下预测因子：

### 不可改变因素

| 因子 | 单位 | 说明 |
|------|------|------|
| 年龄 | 岁 | 主要风险因素 |
| 性别 | - | 男性风险高于女性 |

### 可改变因素

| 因子 | 单位 | 正常值 | 说明 |
|------|------|--------|------|
| 收缩压 | mmHg | <120 | 血压水平 |
| 是否降压治疗 | - | 否 | 正在使用降压药 |
| 总胆固醇 | mmol/L | <5.18 | 血脂水平 |
| HDL-C | mmol/L | ≥1.04 | 好胆固醇 |
| 是否有糖尿病 | - | 否 | 已诊断糖尿病 |
| 是否吸烟 | - | 否 | 当前吸烟 |
| BMI | kg/m² | 18.5-23.9 | 体重指数 |
| 腰围 | cm | 男<85,女<80 | 中心性肥胖（可选） |

## 计算方法

### 模型公式

China-PAR模型基于Cox比例风险回归模型：

```
风险得分 = Σ(βi × Xi)
10年风险概率 = 1 - S₀(t)^exp(个人风险得分 - 平均风险得分)
```

其中：
- βi：各预测因子的回归系数
- Xi：各预测因子的实际值
- S₀(t)：基础生存函数

## 风险分层

### 分层标准

根据10年ASCVD发病风险概率，将人群分为三个风险层级：

| 风险等级 | 10年ASCVD风险 | 管理策略 |
|---------|--------------|---------|
| 低风险 | <5% | 生活方式干预 |
| 中风险 | 5%-9.9% | 生活方式干预+风险评估 |
| 高风险 | ≥10% | 强化干预+药物治疗评估 |

## 临床应用

### 适用人群
- 20岁及以上无心血管病史的成年人
- 用于心血管病一级预防风险评估

### 不适用人群
- 已确诊心血管病（心梗、脑卒中）
- 年龄<20岁
- 数据严重缺失

### 临床决策支持

#### 低风险（<5%）
- 健康生活方式建议
- 定期体检（每2-3年）
- 无需药物治疗

#### 中风险（5%-9.9%）
- 强化生活方式干预
- 定期评估（每年1次）
- 考虑药物治疗（如有多个危险因素）

#### 高风险（≥10%）
- 积极生活方式干预
- 密切监测（每3-6个月）
- 评估药物治疗适应症
- 阿司匹林一级预防（综合考虑）

### 风险降低策略

通过干预可改变因素，可显著降低心血管风险：

| 干预措施 | 预期风险降低 |
|---------|------------|
| 戒烟 | 30-40% |
| 降压治疗 | 25-35% |
| 降脂治疗 | 20-30% |
| 血糖控制 | 15-25% |
| 减重 | 10-20% |

## 模型验证

### 验证研究

China-PAR模型在中国多个人群中进行了验证：
- **训练队列**：中国心血管病流行病学多中心协作研究
- **验证队列**：中国高血压调查、中国代谢综合征研究等
- **C统计量**：男性0.78，女性0.81

### 与其他模型对比

| 模型 | 适用人群 | C统计量 |
|------|---------|---------|
| China-PAR | 中国人群 | 0.78-0.81 |
| Framingham | 美国人群 | 0.70-0.75 |
| SCORE | 欧洲人群 | 0.72-0.78 |

China-PAR模型在中国人群中表现优于其他模型。

## 使用说明

### 计算示例

**患者信息**：
- 男性，55岁
- 血压145/90 mmHg，未治疗
- TC 5.8 mmol/L，HDL-C 1.0 mmol/L
- 无糖尿病，不吸烟
- BMI 26 kg/m²

**计算过程**：
1. 计算风险得分
2. 计算10年风险概率
3. 确定风险等级

**结果**：
- 10年ASCVD风险：8.5%
- 风险等级：中风险
- 同龄人平均风险：6.5%

### 使用脚本计算

```bash
python scripts/china_par_calculator.py \
  --age 55 \
  --gender male \
  --systolic 145 \
  --tc 5.8 \
  --hdl-c 1.0 \
  --bmi 26
```

或从文件计算：
```bash
python scripts/china_par_calculator.py --input validated_data.json
```
